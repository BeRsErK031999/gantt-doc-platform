# gantt-doc-platform

`gantt-doc-platform` — верхнеуровневая document platform для будущей работы с Word, PDF и document flows.

На текущем этапе реализован только foundation-слой: создан минимальный каркас репозитория без зависимостей, без приложений, без реальной функциональности и без раннего усложнения архитектуры.

## Созданные папки

- `apps/api`
- `apps/web`
- `packages/shared-types`
- `docs/architecture`
- `docs/decisions`
- `scripts`

## Что пока не реализовано

- API и web-приложения
- `src`-структуры приложений
- доменные и инфраструктурные слои
- базы данных
- `auth`
- `upload`
- AI-интеграции
- внешние интеграции
- бизнес-логика
- toolchain и package-level настройки

## Подход к развитию

Проект будет развиваться поэтапно. Сначала фиксируется чистый foundation-каркас, затем последующими небольшими шагами будут добавляться структура, конфигурация и функциональные возможности.
